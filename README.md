# BLOOD-CONNECT

**Blood-Connect** — A community-driven blood donor connection web app (frontend + backend).  
(Short tagline: match donors with recipients by blood group and location.)

---

## 🔎 Project Overview
Blood-Connect is a web application built to connect blood donors and recipients in real time. It includes features like user registration, donor search by blood group and proximity, request submission, and basic urgency indicators.

**Status:** Prototype / MVP — work in progress.

---

## 🧭 Features
- User registration & login (Firebase / placeholder)  
- Donor profiles with blood group and location  
- Search donors by blood type & proximity filter  
- Submit blood requests with urgency levels (Low / Moderate / High)  
- Dashboard for donors and hospitals (admin features planned)
